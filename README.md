# student-attendance-monitoring-system

Student Attendance Monitoring System
Overview

A Python-based Student Attendance Monitoring System developed using Tkinter and CSV file handling. The application automates attendance management and reduces manual effort.

Features
Add/Delete Students
Mark Attendance
Save Attendance
View Previous Records
Filter Records
Generate Reports
CSV Storage
User-friendly GUI
Technologies Used
Python
Tkinter
CSV
