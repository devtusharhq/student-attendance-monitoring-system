# 📚 Student Attendance Monitoring System

## 📖 Overview

The **Student Attendance Monitoring System (SAMS)** is a Python-based desktop application developed using **Tkinter** and **CSV file handling**. The system automates the attendance management process, reduces manual effort, minimizes errors, and enables efficient storage and retrieval of attendance records through an intuitive graphical user interface.

---

## ✨ Features

- ➕ Add new students
- 🗑️ Delete existing students
- ✅ Mark student attendance
- 💾 Save attendance records
- 📂 View previous attendance records
- 🔍 Filter records by date, subject, and session
- 📊 Generate attendance reports
- 📁 CSV-based data storage
- 🖥️ User-friendly graphical interface

---

## 🛠️ Technologies Used

- Python
- Tkinter
- CSV File Handling
- Software Engineering (Waterfall Model)
